# 🛒 SmartCart Customer Segmentation

## 📌 Project Overview

SmartCart Customer Segmentation is a Machine Learning project that groups customers into different segments based on their purchasing behavior. The project helps businesses understand customer patterns and create personalized marketing strategies.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**, and is deployed online for real-time predictions.

## ✨ Features

* Predict customer segments instantly
* Interactive Streamlit web application
* Data preprocessing using StandardScaler
* Dimensionality Reduction using PCA
* Customer Segmentation using K-Means Clustering
* User-friendly interface
* Deployable on Streamlit Community Cloud

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/BhoomikaTanwar28/smartcart-customer-segmentation.git
```

Move into the project folder:

```bash
cd smartcart-customer-segmentation
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
